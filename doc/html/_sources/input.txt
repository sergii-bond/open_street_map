Input Data
--------------------------

The map data is of city Kyiv, Ukraine, taken from the Metro extracts on the `MapZen Website <https://mapzen.com/data/metro-extracts>`_

The detailed link to the dataset on the web: `Link <https://s3.amazonaws.com/metro-extracts.mapzen.com/kyiv_ukraine.osm.bz2>`_

Uncompressed XML size : 234M

The reason for this choice of an area is that I was born and raised in Kyiv, the capital of Ukraine.
